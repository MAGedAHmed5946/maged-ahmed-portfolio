Project Overview

As the Unreal Engine Developer on the team, I was responsible for building the entire 3D simulation from the ground up.

  • Integrated ConvAI to power an intelligent, voice-interactive virtual tour guide within the simulation.
  • Structured bidirectional HTTP API communication to synchronize player data seamlessly with a mobile app.
